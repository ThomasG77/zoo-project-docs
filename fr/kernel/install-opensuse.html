OpenSUSE¶

Authors:Nicolas Bozon, Gérald Fenoy, Jeff McKenna
Last Updated:$Date: 2011-12-07 14:44:57 +0100 (Wed, 07 Dec 2011) $

  • OpenSUSE

Le Zoo-Kernel est maintenu en tant que paquet dans OpenSUSE Build Service (OBS). De cette manière, les rpm sont fournis pour toutes les versions de openSUSE Linux (11.2, 11.3, 11.4, Factory).

Versions stables¶

Pour installer le Zoo-Kernel dans openSUSE, il y a 3 manières disponibles:

Installateur en un clic¶

L’installateur en un clic peut être trouvé ici. Pour openSUSE 11.4, c’est le lien direct.

Le gestionnaire de paquets YAST utilisant une interface graphique¶

Le dépôt Application:Geo doit être ajouté aux dépôts logiciels et ensuite, le Zoo-kernel peut être trouvé dans le gestionnaire de logiciels via une recherche.

Ligne de commande (en tant que root pour openSUSE 11.4)¶

zypper ar http://download.opensuse.org/repositories/Application:/Geo/openSUSE_11.4/
zypper refresh
zypper install zoo-kernel

Version instable¶

La dernière version de développement du ZOO-Kernel peut être trouvée dans OBS sous le projet home:tzotsos. Les paquets du ZOO-Kernel sont maintenus et testés ici avant d’être mis à disposition sur le dépôt Application:Geo.

Les méthodes d’installation sont identiques à la version stable. Assurez-vous d’utiliser ce dépôt à la place.

Installation en ligne de commande (en tant que root pour openSUSE 11.4)¶

zypper ar http://download.opensuse.org/repositories/home:/tzotsos/openSUSE_11.4/
zypper refresh
zypper install zoo-kernel
zypper install zoo-kernel-grass-bridge

En plus, il y a l’option d’ajouter le paquet zoo-wps-grass-bridge. Cette option installera automatiquement grass7 (svn trunk).

Essayer l’installation¶

  • http://localhost/cgi-bin/zoo_loader.cgi?ServiceProvider=&metapath=&Service=WPS&Request=GetCapabilities&Version=1.0.0
  • http://localhost/cgi-bin/zoo_loader.cgi?ServiceProvider=&metapath=&Service=WPS&Request=DescribeProcess&Version=1.0.0&Identifier=HelloPy
  • http://localhost/cgi-bin/zoo_loader.cgi?ServiceProvider=&metapath=&Service=WPS&Request=Execute&Version=1.0.0&Identifier=HelloPy&DataInputs=a=myname
