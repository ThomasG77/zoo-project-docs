Service ZOO GDAL¶

Le Service ZOO GDAL est basé sur le code source et le copyright de GDAL. Ce service ZOO a pour but de fournir certaines opérations de base de traitement raster à votre installation du Kernel ZOO. Apprenez en plus et lisez la documentation sur le site web GDAL officiel.

Fonctions implémentées¶

Gdal_Grid

créé des grilles régulières depuis des données dispersées

Gdal_Translate

convertit des données raster entre différents formats

Service ZOO OGR¶

Le Service ZOO OGR est basé sur le code source et le copyright de OGR. Ce service ZOO a pour but de fournir certaines opérations spatiales vecteur de base à votre installation du Kernel ZOO. Apprenez en plus et lisez la documentation sur le site web OGR officiel.

Fonctions implémentées¶

Base_Vect_Ops
